BrewBuddy-AI
BrewBuddy AI is a smart coffee discovery platform that helps users find top cafes, explore specialty beans, and improve brewing techniques. Powered by AI, it offers personalized recommendations, maps integration, and a modern UI experience.

---

🚀 Overview

BrewBuddy AI** is a modern AI-powered coffee discovery and brewing assistant that helps users find the best cafes, explore specialty coffee beans, and improve their brewing techniques.

It combines **AI recommendation systems**, **location-based cafe discovery**, and **interactive brewing guides** into a seamless experience.

---

 ✨ Features

* 🔍 Discover top cafes in Bhopal
* 🤖 AI-powered coffee recommendation system
* ☕ Detailed coffee profiles (Taste, Roast, Brew methods)
* 🗺️ Integrated Google Maps for location & directions
* 🚕 Ride booking integration (Uber, Ola, Rapido)
* 📊 Brew tracking & personalized profiles
* 💬 AI Chatbot (BrewBot AI)
* 💳 Freemium pricing model
* 📱 Responsive modern UI

---

🧠 AI Capabilities

* Taste preference analysis
* Smart bean matching algorithm
* Conversational chatbot (multi-model powered)
* Personalized brewing suggestions

⚙️ Installation

```bash
git clone https://github.com/Adarsh-Singh-Chauhan/brewbuddy-ai.git
cd brewbuddy-ai
npm install
npm run dev
```

---

🌐 Deployment

 Option 1: Vercel (Recommended)

```bash
npm install -g vercel
vercel
```

Option 2: Netlify

```bash
npm run build
```

Upload `/dist` folder to Netlify

---
 🔐 Environment Variables

Create `.env` file:

```env
VITE_OPENAI_API_KEY=your_key
VITE_GOOGLE_MAPS_KEY=your_key
```
